# HR Recruitment Grabber - Chrome Extension

**PENTING: EKSTENSI INI 100% GRATIS DAN TIDAK PERLU MEMBAYAR.**
Anda **TIDAK PERLU** mendaftar sebagai Chrome Developer atau mengunggahnya ke Chrome Web Store (yang memang berbayar $5). Ekstensi ini dibuat khusus untuk digunakan secara "Lokal" di komputer Anda sendiri secara gratis.

Ikuti petunjuk di bawah ini dengan teliti untuk memasangnya secara gratis.

## Cara Mengunduh & Menginstal (Gratis)

1. Di Project ini (AI Studio), klik tab panah bawah (Settings/Pengaturan) di pojok kanan atas layar AI Studio atau Code Editor, lalu pilih **Export to ZIP** / **Download ZIP**.
2. Ekstrak (Unzip) file ZIP yang sudah didownload ke komputer Anda.
3. Buka browser **Google Chrome** di komputer / laptop Anda.
4. Ketik atau copy-paste URL ini di address bar Chrome lalu Enter: `chrome://extensions/`
5. Di pojok kanan atas halaman tersebut, hidupkan saklar **Developer mode** (Mode Pengembang).
6. Akan muncul beberapa tombol baru di kiri atas. Klik tombol **Load unpacked** (Muat yang belum dibongkar).
7. Cari dan pilih folder bernama `chrome-extension` (berada di dalam folder hasil ekstrak langkah ke-2 tadi).
8. Selesai! Ekstensi "HR Recruitment Grabber" akan muncul di daftar ekstensi Chrome Anda dan siap digunakan secara gratis.

## Cara Penggunaan (Penting!)
Karena tampilan web dari Glints/Pintarnya/Jobstreet sering berubah-ubah, ekstensi ini menggunakan sistem **"Tarik dari teks yang disorot (di-block/highlight)"**.

**Langkah-langkahnya:**
1. Buka web rekrutmen seperti **Glints**, **Pintarnya**, **Indeed**, atau **Jobstreet**.
2. **KUNCI UTAMA:** Gunakan mouse Anda untuk MEM-BLOK (highlight/sorot) nama kandidat beserta nomor teleponnya di layar web tersebut (seperti biasa saat mau copy paste).
3. Setelah teks tersebut tersorot biru, barulah **Klik Icon Ekstensi "HR Recruitment Grabber"** di kanan atas Chrome Anda.
4. Voila! Popup akan muncul dan kolom Nama & Nomor Whatsapp akan **Otomatis Terisi** berdasarkan teks yang Anda blok tadi. 
5. Ketik nama Loker yang sedang dilamar (misal: `Driver Gudang`). Harus sama persis dengan loker di Web HR.
6. Pastikan kolom **URL HR Workspace** tertulis `https://hrdhikemore.vercel.app/`.
7. Klik tombol biru **"Kirim ke HR Workspace →"**.
8. Web HR Workspace akan otomatis terbuka di tab baru, langsung loncat ke menu Rekrutmen, membuka loker tersebut, dan memunculkan popup Tambah Kandidat (Penjadwalan WA) lengkap dengan datanya! Tinggal pencet *Simpan*.
